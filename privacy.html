Privacy Policy

PDFPRO is designed to keep document processing simple and privacy-focused. This page explains what we collect and how we use it.

What We Store

We store activity logs for PDF operations such as merge and split. These logs may include the operation type, file names, file sizes, processing time, status, IP address, and browser/device information.

What We Do Not Store

We do not permanently store the full contents of your PDF files through the logging feature shown in this project. The current logging system is intended for operation tracking and debugging.

How Logs Are Used

  • To monitor whether PDF operations succeed or fail
  • To help debug issues in the application
  • To understand basic usage patterns and performance

Your Choice

If you want stricter privacy, you can disable logging in the application code or clear the logs table in your database at any time.
